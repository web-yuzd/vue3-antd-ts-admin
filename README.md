# vue3-antd-ts-admin

基于 vue3、ant-design-vue、typescript 搭建的后台管理系统

# 技术栈

```
vue3
typescript
vue-router
vuex
ant-design-vue
```

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
